# Teacher Activity Diagrams (Code-Accurate, ASCII Style)

## 1) Teacher Login Flow
```
+---------------------------+
| Start                     |
+---------------------------+
            |
            v
+---------------------------+
| Open Teacher Login        |
+---------------------------+
            |
            v
+---------------------------+
| Enter Phone + Password    |
+---------------------------+
            |
            v
+---------------------------+
| Model Valid?              |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Find Teacher by Phone     |   | Show Validation Errors    |
| and Role=Teacher          |   +---------------------------+
+---------------------------+               |
      |                                     +--> back to input
      v
+---------------------------+
| Password Correct?         |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Create Session            |   | Show Invalid Credentials  |
+---------------------------+   +---------------------------+
            |                               |
            v                               +--> back to input
+---------------------------+
| Approved Teacher?         |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Redirect Dashboard        |   | Show Pending Approval     |
+---------------------------+   +---------------------------+
            |                               |
            +---------------+---------------+
                            v
+---------------------------+
| End                       |
+---------------------------+
```

```
+---------------------------+

+---------------------------+
            |
            v

            |
            v
+---------------------------+

            |
            v
+---------------------------+

+---------------------------+
            |
            v
+---------------------------+

      | Yes                       | No

            |                               |
            v                               +--> back to input
+---------------------------+
| Save Material Pages       |
+---------------------------+
            |
            v
+---------------------------+

+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Open Direct Question      |   | Keep Page Without Quiz    |
| Entry (no title input)    |   +---------------------------+
+---------------------------+               |
            |
            v
+---------------------------+

+---------------------------+
      |                           |
      +------------+--------------+
                   v
+---------------------------+

+---------------------------+
```
