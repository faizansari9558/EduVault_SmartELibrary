# Admin Activity Diagrams (Code-Accurate, ASCII Style)

## 1) Admin Login Flow
```
+---------------------------+
| Start                     |
+---------------------------+
            |
            v
+---------------------------+
| Open Admin Login          |
+---------------------------+
            |
            v
+---------------------------+
| Enter Phone + Password    |
+---------------------------+
            |
            v
+---------------------------+
| Model Valid?              |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Find Admin User by Phone  |   | Show Validation Errors    |
| and Role=Admin            |   +---------------------------+
+---------------------------+               |
      |                                     +--> back to input
      v
+---------------------------+
| Password Correct?         |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Create Session            |   | Show Invalid Credentials  |
| (UserId, Role, Name)      |   +---------------------------+
+---------------------------+               |
      |                                     +--> back to input
      v
+---------------------------+
| Redirect Admin Dashboard  |
+---------------------------+
            |
            v
+---------------------------+
| End                       |
+---------------------------+
```
