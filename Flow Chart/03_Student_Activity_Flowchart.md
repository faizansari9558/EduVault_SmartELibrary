# Student Activity Diagrams (Code-Accurate, ASCII Style)

## 1) Student Login Flow (OTP)
```
+---------------------------+
| Start                     |
+---------------------------+
            |
            v
+---------------------------+
| Open Unified Login        |
| (/Login)                  |
+---------------------------+
            |
            v
+---------------------------+
| Enter Phone + Password    |
+---------------------------+
            |
            v
+---------------------------+
| Model Valid?              |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Student Role and          |   | Show Validation Errors    |
| Password Correct?         |   +---------------------------+
+---------------------------+               |
      | Yes                       | No      +--> back to input
      v                           v
+---------------------------+   +---------------------------+
| Route to Student OTP      |   | Show Invalid Credentials  |
| Verification              |   +---------------------------+
+---------------------------+               |
            |                               +--> back to input
            v
+---------------------------+
| Generate OTP and Redirect |
| Verify OTP                |
+---------------------------+
            |
            v
+---------------------------+
| Enter OTP                 |
+---------------------------+
            |
            v
+---------------------------+
| OTP Valid and Session     |
| Start OK?                 |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Sign In Student and       |   | Show OTP/Session Error    |
| Open Dashboard            |   +---------------------------+
+---------------------------+               |
            |                               +--> back to OTP
            v
+---------------------------+
| End                       |
+---------------------------+
```

```
+---------------------------+

            v

            |
            v

+---------------------------+   +---------------------------+

            |                               +--> back to input
            v
+---------------------------+

+---------------------------+
            |
            v
+---------------------------+

+---------------------------+
            |
            v

      | Yes                       | No
```

## 7) Forgot Password Flow (Student)
```
+---------------------------+
| Start                     |
+---------------------------+
            |
            v
+---------------------------+
| Open Forgot Password      |
| (/Auth/ForgotPassword)    |
+---------------------------+
            |
            v
+---------------------------+
| Enter Phone Number        |
+---------------------------+
            |
            v
+---------------------------+
| Model Valid?              |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Phone Exists?             |   | Show Validation Errors    |
+---------------------------+   +---------------------------+
      | Yes                       | No            |
      v                           v               +--> back to input
+---------------------------+   +---------------------------+
| Generate OTP + Save       |   | Show Phone Not Found      |
| ResetPasswordPhone        |   | Error                     |
+---------------------------+   +---------------------------+
            |                               |
            v                               +--> back to input
+---------------------------+
| Redirect Verify OTP       |
| (/Auth/VerifyForgot...)   |
+---------------------------+
            |
            v
+---------------------------+
| Enter OTP                 |
+---------------------------+
            |
            v
+---------------------------+
| OTP Valid?                |
+---------------------------+
      | Yes                       | No
      v                           v
+---------------------------+   +---------------------------+
| Open Reset Password Page  |   | Show Invalid/Expired OTP  |
+---------------------------+   +---------------------------+
            |                               |
            v                               +--> back to OTP
+---------------------------+
| Enter New Password +      |
| Confirm                   |
+---------------------------+
            |
            v
+---------------------------+
| Update PasswordHash and   |
| Clear Temp Phone          |
+---------------------------+
            |
            v
+---------------------------+
| Show Success and Redirect |
| Home                      |
+---------------------------+
            |
            v
+---------------------------+
| End                       |
+---------------------------+
```
